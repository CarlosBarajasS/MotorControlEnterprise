📌 Instrucciones de despliegue

  1. Agrega la línea {{ edgeConfig().mosquittoLine }} al archivo password_file de Mosquitto y reinicia el servicio.
  2. Descarga los 3 archivos y cópialos a la Raspberry Pi dentro de la carpeta edge-gateway/.
  3. Ejecuta docker compose up -d para iniciar el gateway.
  4. Verifica en Dashboard → Cámaras que las cámaras aparezcan en línea (~2 min).
